# Attendance Management System

Full-stack web application to manage user attendance and documents.

## Tech Stack
- Frontend: Next.js
- Backend: NestJS
- Database: Postgres

## Features
- User authentication
- Attendance tracking
- File management

## Installation

### Backend
```bash 
cd attendance-mgmt-back
npm install
npm run start:dev`
```
### Frontend
```bash
cd attendance-mgmt-front 
npm install
npm run dev
```
